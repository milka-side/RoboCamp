# Функції керування машинкою

Це список готових функцій, які можна викликати з `loop()`, щоб керувати машинкою.
Не потрібно розбиратись, як вони працюють всередині — просто викликай і все.

---

## 🚗 Рух

```cpp
Car_SetMotors(int left, int right);
```
Задає швидкість лівого і правого боку машинки.
- Діапазон: **-4095 (назад) до +4095 (вперед)**.
- `0` — стоп.
- Приклад: `Car_SetMotors(2000, 2000);` — їхати вперед прямо.
- Приклад: `Car_SetMotors(2000, -2000);` — крутитись на місці.

---

## 📯 Клаксон

```cpp
Car_SetHorn(bool on);
```
Вмикає/вимикає звуковий сигнал (бузер).
- `Car_SetHorn(true);` — гудок увімкнено.
- `Car_SetHorn(false);` — гудок вимкнено.

---

## 💡 Світлодіоди (стрічка з 12 діодів)

```cpp
LED_SetColor(int led_number, int r, int g, int b);
```
Задає колір ОДНОГО діода.
- `led_number`: від 0 до 11.
- `r`, `g`, `b`: яскравість кольору від 0 до 255.
- Приклад: `LED_SetColor(0, 255, 0, 0);` — перший діод стає червоним.

```cpp
LED_SetAllColors(int r, int g, int b);
```
Задає один колір одразу для всіх діодів.
- Приклад: `LED_SetAllColors(0, 255, 0);` — всі діоди зелені.

```cpp
LED_TurnOff(int led_number);
```
Вимикає один діод.

```cpp
LED_TurnOffAll();
```
Вимикає всі діоди.

## 🖥️ Матриця-екран (очі/емоції)

### Статичні картинки

```cpp
Matrix_Clear();            // очистити екран
Matrix_ShowArrowLeft();     // стрілка вліво
Matrix_ShowArrowRight();    // стрілка вправо
Matrix_ShowStaticEmotion(int id); // показати емоцію за номером (0, 1, 2, ...)
```

### Анімації (продовжують гратись самі, поки не викличеш іншу)

```cpp
Matrix_AnimRotate();      // очі обертаються
Matrix_AnimBlink();       // моргання
Matrix_AnimSmile();       // усмішка
Matrix_AnimCry();         // плач
Matrix_AnimWheelLeft();   // коліщатко крутиться вліво
Matrix_AnimWheelRight();  // коліщатко крутиться вправо
```

⚠️ Щоб анімації насправді рухались кадр за кадром, треба викликати
`Matrix_Update();` в кожному циклі `loop()`. Сама функція просто "вмикає"
потрібну анімацію, а `Matrix_Update()` малює наступний кадр, коли настав час.

## 🔄 Серво ("голова" машинки)

```cpp
Car_SetServo(int angle);
```
Повертає серво на заданий кут.
- Діапазон: **0–180 градусів**. 90 — це центр (дивиться прямо).

## 🎮 Кнопки геймпада

Перед використанням вкажіть колір геймпада у файлі Car_Core.h вказавши правильний тип у визначенні `#define CONTROLLER_TYPE` `CONTROLLER_TYPE_GREY` - для сірого, `CONTROLLER_TYPE_BLUE` - для синього 

```cpp
Btn_Cross(pad)     // хрестик / X
Btn_Circle(pad)    // кружечок
Btn_Square(pad)    // квадрат
Btn_Triangle(pad)  // трикутник

Btn_L1(pad)  Btn_R1(pad)   // верхні кнопки
Btn_L2(pad)  Btn_R2(pad)   // тригери

Btn_L3(pad)  Btn_R3(pad)   // клік стіків

Btn_Start(pad)
Btn_Select(pad)

Btn_DpadUp(pad)
Btn_DpadDown(pad)
Btn_DpadLeft(pad)
Btn_DpadRight(pad)
```

Кожна повертає `true`, якщо кнопка зараз натиснута, і `false`, якщо ні.

Приклад:
```cpp
if (Btn_Cross(pad)) {
  Car_SetHorn(true);
}
```

> **УВАГА:** на сірому контролері наступні кнопки та функції відповідно не працююють: `Btn_Square(pad)`, `Btn_L2(pad)`, `Btn_R2(pad)`, `Btn_DpadRight(pad)`, `Btn_R3(pad)`
